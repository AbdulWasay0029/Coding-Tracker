# CodeSync Screen & Prototype Directives

Generate a fully responsive, dark-themed React UI based on the provided Global Context, Feature Plan, and Style Guide. 

## Prototyping Goals:
1. **Developer Aesthetic:** Focus heavily on creating a highly premium, developer-focused aesthetic that feels like a next-generation dev tool. 
2. **Data Visualization:** Prioritize creative, clean ways to visualize the contribution heatmap and the leaderboard table. They should not look like boring standard tables or grids.
3. **Modularity:** Generate modular React/Vite components that can easily be mapped into our existing Next.js App Router structure.
4. **Interactivity:** Include states for hover, focus, and "locked" vs "unlocked" badges. Include placeholder animations if the prototyping tool supports it.
